# boxd

`boxd` is the single CLI that installs sandbox kits from git repos and runs sbx sandboxes with them. See `CONTEXT.md` for the project's domain glossary and `docs/adr/` for past architectural decisions.

## Agent skills

### Issue tracker

Issues live as markdown files under `docs/.issues/<feature-slug>/`. See `docs/agents/issue-tracker.md`.

### Triage labels

Default canonical role names (`needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`). See `docs/agents/triage-labels.md`.

### Domain docs

Single-context: `CONTEXT.md` + `docs/adr/` at the repo root. See `docs/agents/domain.md`.
